# Lamaati Hair Care Website

Lamaati is a simple React hair care ecommerce website.  
The name Lamaati means “my shine”. The website helps users browse hair care products and place a simple order.
## Project Description

This project is a frontend web application built using ReactJS. It includes several pages for a hair care brand, such as Home, Shop, About, Contact, and Checkout.

The website displays hair care products like scalp oil, hair serum, hair mask, and hair mist. Users can view product details, contact the brand, and fill in a checkout form.

## Setup Instructions

To run the project, open the project folder in VS Code and write these commands in the terminal:

```bash
npm install
npm run dev
```

Then open the local link shown in the terminal, usually:

```text
http://localhost:5173/
```
